# is-divisible-by-3

is-divisible-by-3 is a js library to check if a number is divisible by 3 or not 

## Installation

Use the npm package manager to install

```bash
npm intsall is-divisible-by-3
```

## Usage

```
const {isDivisibleBy3} = require("isDivisibleBy3")

// use it
 isDivisibleBy3(9)

```
